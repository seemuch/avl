AVL
===

How to build
============

1. Make sure you have [Autotools](http://www.gnu.org/software/automake/)
   installed. You can install it using [MacPorts](http://guide.macports.org) on
   Mac or apt-get on Linux.

2. Create configure script.

    autoreconf --install

3. Create Makefiles.

    ./configure

4. Make.

    make
